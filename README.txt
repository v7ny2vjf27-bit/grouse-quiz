【GitHub Pagesで使う方法】
1. ZIPを解凍します。
2. 中のファイルをすべて GitHub の grouse-quiz リポジトリ直下にアップロードします。
3. 既存の index.html / questions.js は上書きしてください。
4. Settings → Pages が main / root になっていれば、そのままURLで動きます。

【注意】
問題データは学習用のたたき台です。法改正・最新判例・試験年度の4月1日現在施行法令に合わせて、必要に応じてe-Gov法令検索等で確認してください。
